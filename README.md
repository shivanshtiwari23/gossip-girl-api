# Gossip Girl API

REST API powering the Gossip Girl anonymous social platform.

Built with Spring Boot and MySQL, this backend handles anonymous post creation, image uploads, comment threads, feed generation, and content persistence for the Gossip Girl web application.

## Features

- Anonymous post creation
- Cloudinary-powered image storage
- Threaded comments and discussions
- Hot feed ranking algorithm
- New feed sorting
- RESTful API architecture
- MySQL database integration
- Spring Data JPA persistence layer
- JSON-based client communication
- CORS support for frontend integration

## Tech Stack

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Cloudinary
- Maven

## Project Overview

The Gossip Girl API serves as the central backend for the Gossip Girl platform. It stores and retrieves user-generated content, manages image uploads through Cloudinary, and exposes endpoints consumed by the frontend application.

The architecture is designed around a clear separation between frontend and backend, allowing the client application to be deployed independently while maintaining a scalable and portable server-side infrastructure.
